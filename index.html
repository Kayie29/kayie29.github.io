Projects

  • Post-Harvest Loss Analysis

    This project explores the causes of post-harvest losses among smallholder farmers in Nigeria. Using collected data, I identified critical factors such as inadequate storage and poor access to transport. The project offered recommendations for tech-enabled interventions like shared cold rooms and SMS-based supply chain alerts.

  • Community WiFi Impact Dashboard

    I created an interactive dashboard to track and visualize the reach and performance of Afrieta’s free internet services in schools and faith-based institutions. Using Excel and Google Data Studio, the dashboard provides real-time visibility into community connectivity patterns and helps prioritize underserved areas.

  • Customer Sentiment Tracker

    This Python-based tool processes customer support chat logs to analyze satisfaction trends. By applying natural language processing basics, the tracker helps pinpoint recurring service issues, enabling timely responses and boosting customer experience.
